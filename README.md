# Siege Night

Escalating siege nights for Project Zomboid (Build 42).

Wave-based attacks, directional hordes, special zombies (Sprinters, Breakers, Tanks), noise-triggered mini-hordes, kill tracking, and full siege history.

Built for fortress defense — solo or multiplayer.

## Features

- **Wave system:** WAVE > TRICKLE > BREAK cycles that escalate through the night
- **Directional attacks** from a random compass direction each siege
- **Special zombies** after midnight (Sprinters blend in, Breakers wear construction gear, Tanks in military outfits)
- **Noise-triggered mini-hordes** between siege nights based on your activity
- **20+ sandbox options** across 3 pages to tune everything
- **Kill tracking + siege history** panel in character info
- **Debug tools** (Numpad keys) for testing

## Install

Subscribe on [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3669589584), or clone this repo into your `Zomboid/mods/` folder.

## Requirements

- Project Zomboid Build 42.14+
